# 施工现场消防安全管理

## 1. 消防责任制度

### 1.1 消防责任人
- 项目经理为消防安全第一责任人
- 设立专职消防安全员
- 建立消防安全管理组织

### 1.2 消防责任制
- 明确各岗位消防责任
- 签订消防安全责任书
- 定期进行消防检查

## 2. 动火作业管理

### 2.1 动火证制度
- 动火作业必须办理动火证
- 动火证当日有效
- 注明动火地点、时间、人员

### 2.2 动火前检查
- 清除周边可燃物
- 准备足够的灭火器材
- 设置防火监护人
- 检查气瓶和管路安全性

### 2.3 动火中监控
- 监护人全程在场监护
- 发现异常立即停止作业
- 保持灭火器在有效范围

### 2.4 动火后检查
- 作业完毕后清理现场
- 确认无火源残留
- 监护至少30分钟

### 2.5 禁止动火情况
- 五级以上大风天气
- 相邻区域有易燃物
- 消防设施不齐全

## 3. 易燃易爆品管理

### 3.1 存放要求
- 设立专门危险品仓库
- 分类存放，标识清晰
- 保持通风，远离火源

### 3.2 使用管理
- 严格执行领用登记
- 限量领用，用不完及时归还
- 使用现场配备灭火器材

### 3.3 废弃物处理
- 废油、废漆单独收集
- 交由专业机构处理
- 严禁随意倾倒

## 4. 消防设施配置

### 4.1 灭火器配置
- 每50㎡至少配置2具灭火器
- 重点部位增加配置密度
- 灭火器类型：ABC干粉灭火器

### 4.2 消防水源
- 设置消防水池或接入市政管网
- 保证消防水压和水量
- 消防栓标识清晰，通道畅通

### 4.3 消防通道
- 保持消防通道畅通
- 设置消防通道标识
- 严禁占用消防通道

## 5. 临时用电管理

### 5.1 线路敷设
- 使用合格电缆
- 架空敷设或穿管保护
- 避免与金属构件摩擦

### 5.2 配电箱
- 使用标准配电箱
- 落实漏电保护措施
- 定期检查维护

### 5.3 防火措施
- 配电箱周围无易燃物
- 配备灭火器材
- 设置防火挡板

## 6. 生活区消防安全

### 6.1 宿舍管理
- 严禁使用明火
- 严禁私拉电线
- 禁止使用大功率电器

### 6.2 厨房管理
- 燃气罐单独存放
- 定期检查燃气管道
- 保持通风良好

### 6.3 吸烟管理
- 设立指定吸烟区
- 配置烟灰缸和水桶
- 严禁随意丢弃烟头

## 7. 消防培训与演练

### 7.1 消防培训
- 新入场人员必须培训
- 定期组织消防知识学习
- 培训记录存档

### 7.2 应急演练
- 每季度组织一次消防演练
- 演练包括报警、疏散、灭火
- 总结演练经验，完善预案

## 8. 应急处置

### 8.1 火灾报警
- 立即拨打119报警
- 报告清楚火灾地点和情况
- 派人到路口引导消防车

### 8.2 人员疏散
- 启动应急疏散预案
- 指定疏散路线和集合点
- 清点人数，确保全部撤离

### 8.3 初期灭火
- 使用灭火器扑灭初期火灾
- 注意自身安全
- 火势无法控制时立即撤离

### 8.4 保护现场
- 火灾后保护现场
- 配合火灾调查
- 总结经验教训
