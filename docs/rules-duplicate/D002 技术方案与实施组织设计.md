# D002 技术方案与实施组织设计

## 检查目标
检查技术路线、实施步骤和组织设计中是否存在非招标强制内容的高度一致表达。

## 重点检查项

### 检查项1：技术路线与步骤
- 比对技术路线的步骤顺序、核心动词、阶段划分及因果关系。
- 若双方章节标题略有调整，但步骤、数字、案例和叙述结构连续一致，应重点标记。

### 检查项2：项目规模及资源数字
- 比对工期、人员数量、设备数量、批次、里程碑和验收比例等数字组合。
- 招标文件未规定的多个数字同时一致时，作为增强证据。

### 检查项3：异常表达
- 检查相同错别字、病句、遗漏编号、复制残留公司名或项目名。
- 检查双方是否出现相同的非标准比喻、口号或内部作业术语。

## 合理重复
- 国家、行业标准名称及其固定条文。
- 招标文件要求原文、技术规范强制参数和通用施工工序名称。
- “项目启动、实施、验收”等缺乏个性的常规阶段名称。

## 不合理重复
- 大段技术论证、步骤顺序、项目数字和异常用词同时一致。
- 一方文本残留另一方公司、人员、系统或项目代号。
- 双方出现相同的非标准编号错误或同一段落跳号。

## 输出要求
- 引用双方完整证据片段和章节位置。
- 区分招标强制重复与自主编制内容重复。
- 相似度必须来自工具候选。
