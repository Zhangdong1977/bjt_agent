# D003 设备材料配置与技术参数

## 检查目标
检查设备、材料、软件模块和技术参数表中非强制内容的相同组合及异常痕迹。

## 重点检查项

### 检查项1：参数组合与表格结构
- 比对设备型号、数量、单位、性能参数、排序及表格列结构。
- 多个非招标强制参数以相同顺序和格式出现时，应重点复核。

### 检查项2：异常型号与录入错误
- 检查不存在或不常见的型号、单位错误、小数点错误及相同错别字。
- 若双方在同一参数上出现相同 OCR 或录入错误，作为疑似不合理重复证据。

### 检查项3：产品描述
- 比对厂家宣传语、定制功能描述和自主选型理由。
- 通用产品名称相同不足以判为疑似重复。

## 合理重复
- 招标文件明确指定的型号、参数、数量和单位。
- 国家标准单位、行业通用设备名称和公共产品规格。
- 厂家公开说明书中的固定参数文本，但需明确其公共来源属性。

## 不合理重复
- 自主选型设备的完整型号、排序、数量及理由均一致。
- 双方具有相同的异常参数、单位错误、型号拼写错误或 OCR 错误。
- 表格中出现相同隐藏备注、内部编码或复制残留。

## 输出要求
- 必须说明参数是否可能来源于招标强制要求或公共产品资料。
- 必须引用双方表格或段落证据及位置。
- 使用工具计算的相似度，不得自行编造。
