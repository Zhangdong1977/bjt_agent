# D004 售后服务与质量保证承诺

## 检查目标
检查售后服务、质量保证、响应时限和违约承诺中具有个性化特征的重复内容。

## 重点检查项

### 检查项1：服务承诺组合
- 比对响应时间、到场时间、修复时间、备件比例和质保期限的组合。
- 招标文件未强制规定而双方数字组合及叙述顺序一致时，应重点标记。

### 检查项2：个性化措辞和错误
- 比对非标准承诺、口号、文言式表达、相同错别字和标点异常。
- 检查双方是否出现相同联系人、电话尾号、邮箱片段或复制残留公司名。

### 检查项3：质量保证流程
- 比对问题受理、升级、闭环和回访流程的步骤与内部岗位名称。
- 相同的内部岗位代号或系统名称属于增强证据。

## 合理重复
- 法规规定、招标文件强制承诺和行业通用服务术语。
- “7×24小时服务”“质量第一”等缺乏独创性的短语。
- 公共厂商标准质保条款，但需有明确公共来源。

## 不合理重复
- 多个自主承诺数字、流程步骤和个性化措辞同时一致。
- 相同联系人痕迹、内部岗位、工单系统名称或复制残留。
- 双方出现相同的罕见错字、病句或非标准法律措辞。

## 输出要求
- 分别引用 A、B 双方证据和位置。
- 说明合理来源或疑似不合理原因。
- 相似度必须使用工具结果。
