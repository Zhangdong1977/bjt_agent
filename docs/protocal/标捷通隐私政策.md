# 标捷通隐私政策

**生效日期：【2026】年【07】月【01】日**
**更新日期：【2026】年【07】月【01】日**

欢迎您使用标捷通！郑州迪维勒普科技有限公司（以下简称“我们”或“迪维勒普”）深知个人信息及商业数据对您的重要性。我们致力于在您使用“标捷通”平台及相关服务（以下简称“本服务”）时，保护您的个人信息与业务数据安全。

本《标捷通隐私政策》（以下简称“本政策”）将帮助您了解我们如何收集、使用、存储、共享和保护您的信息，以及您可以如何管理这些信息。请您在使用本服务前，务必仔细阅读并充分理解本政策。一旦您勾选同意或实际使用本服务，即视为您已同意本政策的全部内容。

### 一、 本政策的适用范围

1.1 本政策适用于您通过网页端、小程序、客户端等形态访问并使用的标捷通各项产品与服务。
1.2 本政策不适用于其他第三方向您提供的服务。当您使用第三方服务时，您的信息处理应适用该第三方的隐私政策，我们对第三方的不当行为不承担法律责任。

### 二、 我们如何收集和使用您的个人信息

为实现标捷通的核心功能（招投标文件解析与审查），我们仅会收集为实现服务目的所必需的信息。

**（一）为实现基础服务功能收集的信息**

1. **注册与登录**：当您注册账号时，我们需要收集您的**手机号码**、短信验证码；如您代表企业注册，我们可能需要收集您的**企业名称**。如您拒绝提供，将无法创建账号及使用本服务。
2. **服务功能实现**：为了提供招投标文件解析与智能检查服务，您需要主动上传**招标文件及投标文件**。系统将提取文件中的项目信息、资格要求、业绩要求等内容进行比对分析。**该类文件属于您的核心业务数据，我们将严格保密。**
3. **计费与充值**：因本服务采用预充值模式，当您进行充值或消耗“文”额度时，我们会收集您的**充值金额、支付时间、支付渠道、交易订单号及账户余额信息**，以保障计费准确及交易安全。
4. **安全与运营保障**：为保障平台安全稳定运行，防范欺诈及网络攻击，当您使用本服务时，我们会自动收集您的**设备信息**（如设备型号、操作系统版本、唯一设备识别码如OAID/IDFA等）、**日志信息**（如访问时间、IP地址、浏览器类型、操作记录）。

**（二）需征得您授权同意的附加功能**

1. **消息通知**：为及时向您推送检查完成状态、系统公告等信息，我们可能会向您收集微信/短信通知授权。您可以在设置中随时关闭该功能。
2. **数据用于模型优化**：**未经您的单独明确授权，我们绝不会将您上传的招投标文件及生成的检查结果用于任何AI大模型的训练与优化。**

**（三）无需征得您授权同意的例外情形**
在以下情形中，我们收集、使用您的个人信息无需征得您的授权同意：

1. 与国家安全、国防安全直接相关的；
2. 与公共安全、公共卫生、重大公共利益直接相关的；
3. 与刑事侦查、起诉、审判和判决执行等直接相关的；
4. 出于维护您或他人的生命、财产等重大合法权益但又很难得到本人同意的；
5. 您自行向社会公众公开的个人信息；
6. 从合法公开披露的信息中收集个人信息的；
7. 法律法规规定的其他情形。

### 三、 我们如何委托处理、共享与披露您的信息

**（一）委托处理与第三方API调用**

1. **大模型服务调用**：标捷通的部分AI解析与检查功能依赖于调用第三方公开的大模型API接口（如DeepSeek等）。为了执行您的指令，我们**必须将您上传的文件内容或提取的文本信息传输至上述第三方大模型服务器进行运算**。
2. **安全保障与约束**：在传输前，我们会尽力采取去标识化或脱敏措施；同时，我们已与相关第三方服务商签署数据处理协议，要求其严格按照我们的指令处理数据，**不得将您的数据用于其自身的模型训练，且在完成运算后予以删除**。但请您理解，第三方服务商对其自身系统的安全性承担独立责任。
3. **支付服务**：为完成充值交易，我们需要将您的支付订单信息共享给第三方支付机构（如微信支付、支付宝等）。
4. **云服务与安全风控**：我们可能委托合规的云服务商进行数据存储，或引入第三方安全风控服务提供商来识别异常账号行为。上述受托方均需签署严格的保密协议。

**（二）转让与公开披露**
原则上，我们不会将您的信息转让或公开披露给任何无关第三方。但在以下情况下除外：

1. 获得您的明确同意；
2. 在合并、收购、资产转让等交易中，如涉及信息转让，我们将要求新主体继续受本政策约束，否则重新征求您的同意；
3. 根据法律法规、诉讼仲裁需要，或应行政、司法机关的强制要求；
4. 为保护标捷通平台、用户及公众的人身财产安全免遭侵害。

### 四、 我们如何保护和保存您的个人信息

**（一）数据安全防护措施**
我们深知招投标文件的机密性，采取了多重措施保护您的数据安全：

1. **加密传输与存储**：采用SSL/TLS协议对网络传输数据进行加密，对存储在服务器中的敏感文件使用AES-256等高强度加密算法。
2. **访问权限管控**：实施严格的内部访问权限控制与操作审计，仅授权经过背景审查且签署保密协议的核心技术人员在故障排查等必要场景下接触数据。
3. **安全监测**：部署防火墙、入侵检测系统等技术手段，防范黑客攻击与数据窃取。

**（二）数据存储期限**
我们仅在为实现本政策所述目的所必需的合理期限内保存您的信息。在您注销账号或主动删除数据后，我们将在30日内从生产系统中清除您的信息；出于系统备份与审计需要，备份系统中的数据将在不超过6个月的期限内被彻底删除或匿名化处理。

### 五、 您如何管理您的个人信息

您对您的个人信息享有完整的控制权。您可以通过平台“个人中心”或通过本政策文末的联系方式行使以下权利：

1. **查阅与复制**：您可随时查阅您的账号信息、充值记录等，并可申请复制相关数据。
2. **更正与补充**：当您的信息发生变化时，您可自行修改或联系我们进行更正。
3. **删除与撤回授权**：您可随时自行删除已上传的招投标文件及生成的检查报告；您亦可随时撤回对我们使用您信息（如消息通知、模型优化）的授权。
4. **注销账号**：您有权随时申请注销标捷通账号。注销后，我们将停止提供服务并依法删除您的个人信息。**请注意，账号注销后，您账户内未消耗的“文”额度将自动清零且不予退款。**

### 六、 未成年人保护

本服务主要面向企业用户及具备完全民事行为能力的成年人。我们不主动收集未成年人的个人信息。若我们发现在未获监护人同意的情况下收集了未成年人的信息，我们将立即采取措施予以删除。

### 七、 本政策的更新

为适应技术发展与法律法规要求，我们可能会适时修订本政策。对于重大变更（如收集目的变更、核心数据处理逻辑调整等），我们将通过弹窗、邮件或短信等显著方式通知您。如您不同意修订后的内容，您有权停止使用本服务；如您继续使用，则视为您接受修订后的政策。

### 九、 附录：术语定义

1. **个人信息**：以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息，不包括匿名化处理后的信息。
2. **设备信息**：指您使用的终端设备基本情况，包括设备型号、操作系统版本、唯一设备识别码（如OAID、IDFA、Android ID）、MAC地址等。
3. **日志信息**：指您访问平台时自动记录的数据，包括访问时间、IP地址、浏览器类型、操作指令等。
4. **去标识化**：指个人信息经过处理，使其在不借助额外信息的情况下无法识别特定自然人的过程。
5. **模型幻觉**：指人工智能大模型在生成内容时，可能产生的与事实不符、逻辑矛盾或捏造信息的现象。
