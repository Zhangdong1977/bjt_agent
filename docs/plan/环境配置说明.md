# 环境配置说明文档

本文档详细说明了标书检查系统开发、测试和生产环境的所有配置信息。


**验证连接**：
```bash
psql -h 183.66.37.186 -p 7004 -U ssirs_user -d bjt_agent -c "SELECT version();"
```

### 2. 网络代理配置

```
HTTP代理: http://127.0.0.1:7890
HTTPS代理: http://127.0.0.1:7890
```

**环境变量设置**：
```bash
export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=http://127.0.0.1:7890
export http_proxy=http://127.0.0.1:7890
export https_proxy=http://127.0.0.1:7890
```

**用途**：
- 访问github
- 访问 Mini-Max API

### 5. Celery Worker 启动

```bash
cd /home/openclaw/bjt_agent
PYTHONPATH=/home/openclaw/bjt_agent \
celery -A backend.celery_app worker \
--loglevel=info -Q review,parser \
--concurrency=4
```

> ⚠️ **注意**: 必须设置正确的 `PYTHONPATH` 和使用 `backend.celery_app` 模块名。
