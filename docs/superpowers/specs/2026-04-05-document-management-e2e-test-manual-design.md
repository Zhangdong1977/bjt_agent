# 文档管理E2E测试手册设计规格

## 1. 概述

本手册旨在提供标书审查智能体系统的文档管理功能端到端测试指南。测试使用Chrome DevTools在VNC远程桌面环境（DISPLAY=:2）中进行，基于Playwright工具录制/执行测试流程。

**测试环境：**
- VNC远程桌面：DISPLAY=:2
- 测试账号：zhangdong / 7745duck
- 测试文档：testdocuments/RTCMS技术规范书.docx

## 2. 测试范围

### 2.1 涵盖功能

| 功能模块 | 测试内容 |
|----------|----------|
| 认证登录 | 使用预设账号登录系统 |
| 文档上传 | 上传招标书、应标书（Word格式） |
| 文档查看 | 查看已解析文档的内容和图片 |
| 文档删除 | 删除已上传的招标书和应标书 |

### 2.2 测试用户流程

```
用户登录 → 创建/选择项目 → 上传文档 → 查看文档 → 删除文档
```

#### 文档查看详细验收标准（DOC-E2E-003/004）

| 验收点 | 描述 |
|--------|------|
| **弹窗显示** | 模态弹窗正确弹出，标题包含文档类型和文件名 |
| **加载状态** | 加载过程中显示"正在加载文档..." |
| **内容呈现** | Markdown/HTML内容正确渲染，无乱码 |
| **图片网格展示** | 图片以网格形式正确展示在文档内容下方 |
| **图片Alt文本** | 每张图片alt属性正确设置（如"页面图片 1"） |
| **图片描述文本** | 每张图片下方显示AI生成的图片内容描述文字 |
| **无内容提示** | 若文档无内容，显示"无内容"提示 |
| **关闭功能** | 点击"×"按钮或遮罩层可关闭弹窗 |

## 4. 测试执行指南

### 4.1 环境准备

1. **启动VNC会话**
   ```bash
   # 确保VNC服务运行在DISPLAY=:2
   ```

2. **启动Chrome DevTools**
   ```bash
   DISPLAY=:2 google-chrome --remote-debugging-port=9222
   ```

3. **访问系统**
   - 打开Chrome浏览器
   - 访问系统URL（如：http://localhost:3000）

### 4.2 执行步骤

1. 打开Chrome DevTools (F12)
2. 切换到Console标签页观察日志输出
3. 按照测试用例步骤执行操作
4. 记录实际结果与预期是否一致
5. 如发现异常，截图并记录错误信息

### 4.3 注意事项

- 测试文档：testdocuments/RTCMS技术规范书.docx
- 每个测试前确认上一个操作已完成
- 文档解析需要等待状态变为"parsed"
- 删除操作不可恢复，请谨慎确认
