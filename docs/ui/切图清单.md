## 四、命名规范说明

1. **页面前缀**：`login`=登录页 / `home`=首页 / `plan`=套餐弹窗 / `common`=通用。
2. **类型**：`input`=输入框前置、`icon`=独立图标、`btn`=按钮、`divider`=分割线、`illustration`=插画、`logo`=Logo。
3. **状态后缀**：`selected`=选中态、`full`=已满态、`black`=黑色版、`theme`=主题色版；默认态不加后缀。
4. **分隔符**：全小写 + 连字符，符合前端资源命名惯例，可直接用于 `import` / CSS class。
