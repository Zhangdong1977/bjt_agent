# Frequently Asked Questions

Find answers to common questions about our platform.

## General Questions

### What is this platform?

Our platform is a comprehensive solution for managing and searching documentation, knowledge bases, and other textual content using advanced AI-powered search.

### How does the search work?

We use hybrid search combining:
- **Vector embeddings** for semantic understanding
- **BM25 keyword search** for precise matches
- **Result fusion** for optimal relevance

This means you can find relevant content even if you don't use the exact words.

### Is my data secure?

Yes. We implement:
- End-to-end encryption
- Secure API key storage
- Regular security audits
- Compliance with SOC 2 and GDPR

### How do I delete my account?

Go to Settings > Account > Delete Account. Note:
- All data will be permanently deleted
- Action cannot be undone
- Final billing will be processed
